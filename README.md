## 주요기능
- 외국인 관광객을 위한 한국인 가이드 매칭 서비스
- 자신의 스펙 및 코스를 웹상에올려 잠시 만나는 가이드 매칭
- 외국인이 자신이 원하는 코스를 등록시 한국인 가이드가 해당 코스 제안 기능

## 주요 기술
- Oracle을 사용하여 DB 설계 및 구축
- AJAX를 이용한 서버 측과 클라이언트측간의 비동기식 데이터 교환 기술
